1. What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications.

2. Basic Structure of HTML Document

An HTML document consists of several elements including <!DOCTYPE html>, <html>, <head>, and <body>.

3. HTML Elements and Tags

HTML elements are represented by tags. Tags are enclosed in angle brackets, like <tagname>.

4. Creating Links in HTML

Links in HTML are created using the anchor tag (<a>). Example: <a href="https://example.com">Visit Example</a>

5. HTML Lists

HTML supports three types of lists: ordered lists, unordered lists, and definition lists.
